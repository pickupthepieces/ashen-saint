# 《灰圣徒》第一章设计：烂骨村

状态：草案  
版本：0.2  
日期：2026-05-09  
依赖文档：`docs/ashen-saint-setting.md` v0.4  
用途：定义第一章剧情、关卡流程、敌人节奏、Boss、掉落和通关后遗境解锁。本文档用于后续 Godot 垂直切片实现。

## 1. 设计目标

| 目标 | 要求 |
|---|---|
| 剧情目标 | 第一章 MUST 完整呈现“黑暗里有救赎”的基调，并建立洛温、米拉、白焰圣堂、王国之伤四条核心线索 |
| 玩法目标 | 第一章 MUST 教会移动、跳跃、普攻、重击、闪避、拾取、装备穿戴、精英怪、Boss 战 |
| 刷子目标 | 第一章通关后 MUST 解锁遗境入口，让玩家进入刷装循环 |
| 工程目标 | 第一章 SHOULD 作为 Godot 首个垂直切片，验证战斗、掉落、背包、场景切换和基础剧情触发 |
| 美术目标 | 第一章 SHOULD 确立灰焰、白灰、破村、旧圣徽、未愈之伤等视觉符号 |

## 4. 核心情绪

第一章不是“屠村爽关”。玩家一开始以为自己在清理怪物，随后发现烂骨村早已被献祭，村民只是被困在遗境中重复死亡当天。

章节结尾 MUST 给出救赎，但不能廉价复活死者。洛温击败赫尔曼后，村民停止重复受难，米拉活下来并成为据点 NPC。烂骨村本身坍缩成遗境，成为后续刷装入口。

## 5. 关键术语

| 术语 | 用法 |
|---|---|
| 王国之伤 | 主灾厄，史书和学者用语 |
| 伤口 | 民间对王国之伤的简称 |
| 北境裂隙 | 白焰圣堂在文书中使用的中性称呼 |
| 灰疫 | 被伤口影响后的污染病症 |
| 灰焰 | 腐化火焰，白焰失败后的残留 |
| 白灰 | 献祭和灰焰烧灼后的残留，可掩盖活人气息 |
| 遗境 | 死者记忆、仪式残留和怪物回声形成的小秘境 |
| 深层 | 伤口更深处，用于后期爬层挑战 |
| 裂口 | 进入遗境或深层的入口 |
| 裂片 | 遗境和深层相关钥匙/材料 |

## 6. 主线流程

| 顺序 | 场景 | 主要目标 | 教学/系统 | 剧情推进 |
|---|---|---|---|---|
| 1 | 灰坑 | 醒来并拿起双手剑 | 移动、跳跃、普攻、拾取 | 洛温从尸灰中复活，胸口有未愈之伤 |
| 2 | 旧麦路 | 穿过村外道路 | 跳跃、重击、闪避、装备掉落 | 发现村民重复献祭当天的动作 |
| 3 | 烂骨村 | 找到幸存者踪迹 | 背包、穿戴、远程怪、快速怪 | 村中已无真正活人，只有仪式残响 |
| 4 | 白木礼拜堂 | 救下米拉 | 精英怪、剧情交互 | 米拉说明村子多次被献祭 |
| 5 | 村心广场 | 击败赫尔曼 | Boss 战、阶段转换 | 让死者停止重复受难，开启裂口 |
| 6 | 通关后据点 | 与米拉交谈 | 遗境入口、装备整理 | 解锁遗境刷装循环 |

## 7. 场景设计

### 7.1 灰坑

| 项目 | 设计 |
|---|---|
| 功能 | 开场复活区和基础移动、跳跃、普攻教学 |
| 视觉 | 尸灰、雨水、半埋墓碑、断剑、烧焦草地 |
| 敌人 | 2-3 个灰化村民 |
| 掉落 | 破旧双手剑、基础胸甲、少量金币 |
| 出口 | 通往旧麦路 |

叙事要求：
- 洛温 MUST 在无旁白的环境中醒来，先看到自己的手和胸口裂伤。
- 灰坑 MUST 有一处低台阶或塌陷墓坑，教学单段跳。
- 玩家拾取“旧裁决剑”的断刃后才能攻击。
- 灰坑尽头 SHOULD 有被烧毁的赦罪庭旗帜，暗示洛温过去身份。

关键文本：

| 触发 | 文本 |
|---|---|
| 醒来 | “雨水落进灰里，露出一只还在动的手。” |
| 拾取武器 | “剑柄被烧黑，护手上还留着赦罪庭的旧印。” |
| 首次受击 | “胸口的伤没有流血，只透出一点暗光。” |

### 7.2 旧麦路

| 项目 | 设计 |
|---|---|
| 功能 | 第一段轻战斗和掉装教学 |
| 视觉 | 烧焦麦田、泥路、枯树、废弃路牌、远处村影 |
| 敌人 | 灰化村民、捧火人 |
| 掉落 | 普通武器、魔法手套、灰焰材料 |
| 出口 | 通往烂骨村 |

玩法要求：
- MUST 使用一处倒塌路障让玩家用跳跃通过。
- MUST 教学重击破防，用一个持农具的灰化村民作为目标。
- MUST 第一次出现捧火人，教玩家躲避远程灰焰。
- SHOULD 在路边放置第一件魔法装备，让玩家看到蓝色品质和词缀。

环境叙事：
- 路牌写着“烂骨村”，但名字被白灰覆盖一半。
- 路边有一具赦罪庭骑士尸体，身上带有封蜡命令。

封蜡命令文本：

> 烂骨村需于入夜前完成清点。  
> 名单不齐者，由村长赫尔曼担保。

### 7.3 烂骨村

| 项目 | 设计 |
|---|---|
| 功能 | 第一章主探索区，强化“重复献祭当天”的异常感 |
| 视觉 | 破木屋、灰焰炉、关着的门、跪拜尸体、白灰堆 |
| 敌人 | 灰化村民、捧火人、缝犬 |
| 掉落 | 魔法装备、金币、白灰、裂片碎屑 |
| 出口 | 通往白木礼拜堂 |

玩法要求：
- MUST 出现缝犬，形成第一波速度压力。
- MUST 有一个小型精英前哨，放置一名赦罪残兵或强化灰化村民。
- SHOULD 设置 2-3 个可互动尸体文本，补充村民不是普通怪物的事实。

尸体文本示例：

| 位置 | 文本 |
|---|---|
| 井边 | “别让孩子出声。白灰能盖住活人的味道。” |
| 炉旁 | “他们说只献一次。村长说他亲眼看过圣堂的文书。” |
| 门后 | “如果我又开始祈祷，求你别听。” |

### 7.4 白木礼拜堂

| 项目 | 设计 |
|---|---|
| 功能 | 剧情反转和米拉登场 |
| 视觉 | 白木梁柱、地下室、刻名墙、蜡油、血迹、破圣徽 |
| 敌人 | 白灰奴、赦罪残兵 |
| 掉落 | 稀有装备首掉、药剂、裂片 |
| 出口 | 通往村心广场 |

剧情要求：
- 米拉 MUST 在地下室出现，不能作为战斗同伴。
- 米拉 MUST 说明白灰可以掩盖活人气息。
- 刻名墙 MUST 展示烂骨村被献祭多次，而不是一次灾难。

米拉首次对白：

| 说话者 | 文本 |
|---|---|
| 米拉 | “别靠近灯。它们会听见你还活着。” |
| 洛温 | “村里还有别人吗？” |
| 米拉 | “有。每晚都有。可他们醒来的时候，已经不是人了。” |

刻名墙文本：

> 第一列是献给白焰的人。  
> 第二列是替他们作证的人。  
> 第三列还是他们。

玩法要求：
- MUST 首次出现赦罪残兵，作为第一章精英怪。
- 赦罪残兵 SHOULD 掉落第一件稀有装备，推动装备筛选体验。
- 米拉被救后 SHOULD 开启短暂护送段，但不要求复杂 AI；她可以在安全点等待玩家清怪。

### 7.5 村心广场

| 项目 | 设计 |
|---|---|
| 功能 | 第一章 Boss 战 |
| 视觉 | 倒塌祭架、灰焰祭坛、尸体缝线、村民名单、白灰风、两侧低台 |
| 敌人 | Boss：赫尔曼，最后的村长 |
| 掉落 | 剧情传奇双手剑部件、裂片、稀有装备 |
| 结果 | 开启裂口，解锁遗境 |

Boss 前演出：
- 洛温和米拉到达村心广场。
- 赫尔曼仍在点名，像活着时一样喊村民名字。
- 每喊一个名字，广场边缘就有灰化者跪下。

Boss 前对白：

| 说话者 | 文本 |
|---|---|
| 赫尔曼 | “名单不能少。少一个，圣堂就不会放过我们。” |
| 米拉 | “他把我们的名字交出去，又把我们一个个喊回来。” |
| 洛温 | “赦免不是这样来的。” |
| 赫尔曼 | “骑士，你比我更懂这个词吗？” |

## 8. Boss 设计：赫尔曼，最后的村长

| 项目 | 设计 |
|---|---|
| 血量阶段 | 两阶段 |
| 定位 | 第一章终点 Boss，考验闪避、重击破绽、清小怪 |
| 尺寸 | 大型横版 Boss，高度约为玩家 2.3 倍 |
| 弱点 | 裂开的胸腔 |
| 主要奖励 | 剧情传奇双手剑部件、裂片、遗境解锁 |

### 8.1 一阶段

| 技能 | 行为 | 应对 |
|---|---|---|
| 重杖横扫 | 近身横向大范围挥击 | 后撤或闪避穿过 |
| 尸手抓取 | 地面伸出尸手定身 | 观察地面白灰预警并移动 |
| 点名 | 召唤 2-3 个灰化村民 | 快速清怪，避免被包围 |
| 下跪 | 短暂防御，减少伤害 | 使用重击破防 |

一阶段 MUST 让玩家理解 Boss 的硬直窗口。赫尔曼每次重杖横扫后 SHOULD 有短暂疲劳，允许玩家反击。

### 8.2 二阶段

触发条件：Boss 生命值低于 50%。

| 技能 | 行为 | 应对 |
|---|---|---|
| 灰焰喷吐 | 胸腔裂开，释放扇形灰焰 | 躲到背后或远离 |
| 白灰风 | 场地两侧周期性出现灰焰带 | 保持中场移动，必要时跳过低矮灰焰 |
| 捧火人召唤 | 召唤 1-2 个自爆捧火人 | 优先击杀或引爆远离 Boss |
| 连续点名 | 连续召唤灰化者并进入低血狂暴 | 保持清怪节奏，等待横扫后反击 |

二阶段 MUST 提升压迫感，但不应变成弹幕。重点是让玩家利用装备、药剂和重击节奏完成战斗。

### 8.3 击败演出

击败后，赫尔曼不应直接爆炸消失。他 SHOULD 重新变回半跪姿态，把村民名单递给米拉或落在地上。

击败后文本：

| 说话者 | 文本 |
|---|---|
| 赫尔曼 | “名单齐了么？” |
| 米拉 | “不用再点了。” |
| 洛温 | “他们可以走了。” |

随后村心广场的跪拜死者逐渐化灰，灰焰熄灭，地面留下裂口。

## 9. 敌人节奏

| 场景 | 敌人组合 | 目的 |
|---|---|---|
| 灰坑 | 少量灰化村民 | 教移动、跳跃、普攻、受击 |
| 旧麦路 | 灰化村民 + 捧火人 | 教重击、闪避、远程威胁 |
| 烂骨村 | 灰化村民 + 捧火人 + 缝犬 | 教走位、聚怪、快速目标处理 |
| 白木礼拜堂 | 白灰奴 + 赦罪残兵 | 教破防、精英怪、稀有掉落 |
| 村心广场 | Boss + 召唤怪 | 综合考验 |

难度要求：
- 第一章 SHOULD 允许普通玩家首通死亡 1-3 次。
- 普通怪不能秒杀玩家。
- 精英怪 MUST 明显比普通怪强，但掉落也必须明显更好。
- Boss 失败后 MUST 从村心广场前安全点重试。

## 10. 装备与掉落

| 阶段 | 掉落要求 |
|---|---|
| 灰坑 | MUST 给玩家基础双手剑和一件普通防具 |
| 旧麦路 | SHOULD 首次掉落魔法装备，词缀数量 1-2 个 |
| 烂骨村 | SHOULD 掉落 2-4 件魔法装备，至少一件与移动或生命相关 |
| 白木礼拜堂 | MUST 首次掉落稀有装备，词缀数量 3 个 |
| Boss | MUST 掉落剧情传奇双手剑部件；SHOULD 额外掉落 1 件稀有装备 |
| 通关后遗境 | MUST 能持续掉落魔法/稀有装备和裂片 |

第一章词缀池：

| 词缀 | 数值范围建议 | 用途 |
|---|---|---|
| 攻击力提高 | 3%-10% | 基础伤害成长 |
| 暴击率提高 | 2%-6% | 重剑构筑雏形 |
| 最大生命提高 | 4%-12% | 容错 |
| 护甲提高 | 4%-12% | 容错 |
| 移动速度提高 | 2%-6% | 刷图手感 |
| 灰焰伤害提高 | 4%-10% | 灰焰构筑雏形 |
| 击杀恢复生命 | 1-5 点 | 清怪续航 |
| 对精英伤害提高 | 5%-12% | Boss 和精英体验 |

剧情传奇部件：

| 项目 | 设计 |
|---|---|
| 名称 | 旧裁决剑：断刃 |
| 部位 | 双手剑部件 |
| 第一版作用 | 作为剧情物品，可在通关后解锁一次武器强化或传奇词缀 |
| 后续扩展 | MAY 在第二章收集剑柄、护手等部件，最终重铸完整武器 |

## 11. 遗境解锁

| 项目 | 设计 |
|---|---|
| 触发 | 击败赫尔曼后，村心广场出现裂口 |
| 首次进入 | 米拉提示玩家裂口中能听见村民残响 |
| 功能 | 进入第一版小秘境：烂骨村遗境 |
| 产出 | 装备、裂片、灰焰材料 |
| 叙事 | 玩家不是单纯刷怪，而是在清理残留痛苦和回收被污染遗物 |

米拉提示：

> 他们没有回来。只是留下了一点声音。  
> 如果你还要进去，别把那些声音当成人。

遗境第一版规则：
- MUST 使用烂骨村主题地图块。
- MUST 随机生成普通怪组合。
- SHOULD 在结尾生成一个精英怪。
- MUST 在结算时给出装备掉落。
- MAY 添加 1 个遗境异象，例如“白灰风：怪物死亡后留下短暂灰雾”。

## 12. 据点解锁

第一章结束后 SHOULD 解锁临时据点“旧营火”。据点不需要完整城镇，但必须支持最小刷子循环。

| NPC/设施 | 第一版功能 |
|---|---|
| 米拉 | 主线对话、遗境提示 |
| 奥德里克 | 鉴定装备、解释词缀 |
| 瑟温修女 | 药剂补给、治疗 |
| 储物箱 | MAY 延后实现 |
| 裂口 | 进入遗境 |

据点基调：
- 不能明亮安全到破坏黑暗感。
- 必须有“活人重新聚在一起”的救赎感。
- 旧营火 SHOULD 使用普通火焰，不是白焰或灰焰。

## 13. 对白原则

| 规则 | 要求 |
|---|---|
| 克制 | 对白 MUST 短句优先，不解释过多设定 |
| 信息密度 | 每段对白 SHOULD 推进人物、真相或任务之一 |
| 命名 | NPC SHOULD 使用民间词“伤口”，教会文本使用“北境裂隙” |
| 洛温 | 洛温说话 SHOULD 冷、少，但不是无情 |
| 米拉 | 米拉 SHOULD 直白，不使用过度诗化语言 |
| 教会文本 | 教会文本 SHOULD 礼貌、正式、残忍 |

示例：

| 来源 | 文本 |
|---|---|
| 教会封蜡文书 | “为免灰疫外溢，烂骨村全员需于白焰前登记。” |
| 米拉 | “他们让我们排队。说排到名字的人会先被救。” |
| 洛温 | “我以前也说过这种话。” |
| 赫尔曼 | “我只交出名单。火不是我点的。” |

## 14. 实现验收

| 验收项 | 标准 |
|---|---|
| 主线可通关 | 玩家能从灰坑一路打到赫尔曼，并完成第一章 |
| 基础战斗 | 玩家能移动、跳跃、普攻、重击、闪避、受击、死亡、重试 |
| 掉落生效 | 普通、魔法、稀有装备能掉落并穿戴，至少 6 个词缀能改变属性 |
| Boss 完整 | 赫尔曼有两阶段、召唤、场地压力和击败演出 |
| 剧情触发 | 米拉登场、刻名墙、Boss 前后对白能触发 |
