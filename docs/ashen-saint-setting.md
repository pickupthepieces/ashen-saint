# 《灰圣徒》设定表

状态：草案  
版本：0.4  
日期：2026-05-09  
用途：作为《灰圣徒》剧情、角色、敌人、美术模型、装备命名和后续 Godot 工程实现的基准文档。

## 1. 核心定位

| 项目 | 设定 |
|---|---|
| 游戏名 | 《灰圣徒》 |
| 英文名 | Ashen Saint |
| 类型 | 横版 2D 暗黑风刷子 ARPG |
| 战斗视角 | 带跳跃的横版 2D，不做复杂跳台关卡；强调站位、技能、聚怪、纵向躲避和装备构筑 |
| 第一版目标 | 第一章剧情 Demo + 遗境刷装原型 + 5 层深层挑战原型 |
| 长期目标 | 完整主线剧情通关后，进入遗境刷词缀、深层挑战爬层的长期循环 |
| 基调 | 黑暗里有救赎；世界残酷，但主角的行动必须带来真实改变 |
| 核心爽点 | 前期靠剧情和战斗推进，后期靠装备筛选、词缀构筑和成型后的爽快割草 |

## 2. 创作约束

| 规则 | 要求 |
|---|---|
| 原创性 | MUST 使用原创世界观、人物、阵营、怪物和装备名；MUST NOT 直接使用《暗黑破坏神》系列角色名、阵营名、地名或具体设定 |
| 暗黑风格 | SHOULD 借鉴暗黑宗教、献祭、腐败、末世村镇、沉重战斗的气质，但 MUST 形成自己的视觉符号 |
| 故事方向 | MUST 保持“黑暗里有一点救赎”的正向基调；MUST NOT 做成纯虚无、纯折磨或毫无希望的叙事 |
| 命名风格 | SHOULD 克制、自然、像真实历史和民间传闻；MUST 避免页游感、堆酷词和过密专有名词 |
| 模型设计 | MUST 先保证剪影辨识度，再考虑细节；角色、怪物和 Boss 都 SHOULD 有清晰轮廓特征 |
| 第一版范围 | MUST 控制在可实现的垂直切片内；完整主线、完整终局、多职业、多套装 MAY 后续扩展 |

## 3. 世界观

| 项目 | 设定 |
|---|---|
| 世界/王国 | 维尔霍恩 |
| 主灾厄 | 王国之伤 |
| 信仰体系 | 白焰信仰 |
| 主要教会 | 白焰圣堂 |
| 审判机构 | 赦罪庭 |
| 民间称呼 | 伤口 |
| 教会文书称呼 | 北境裂隙 |
| 污染病症 | 灰疫 |
| 腐化火焰 | 灰焰 |
| 异空间 | 遗境、深层 |
| 被污染者统称 | 灰化者 |

维尔霍恩曾信奉白焰，认为火可以净化罪、疫病和恶念。后来王都北方出现一道无法闭合的裂隙，灰焰从地下涌出，灰疫随风扩散。民间称它为“伤口”，史书称它为“王国之伤”，白焰圣堂在文书中称它为“北境裂隙”。

白焰圣堂对外宣称裂隙是神明试炼，实际用献祭拖延伤口扩张。献祭失败后，死者记忆、仪式残留和怪物回声形成遗境；伤口更深处则被称为深层。遗境和深层是后期刷装与爬层的叙事来源。

## 4. 主角设定

| 项目 | 内容 |
|---|---|
| 姓名 | 洛温·阿什 |
| 称号 | 灰圣徒；“被焚者”只作为少量剧情称呼使用 |
| 职业原型 | 双手剑近战战士 |
| 曾经身份 | 白焰圣堂赦罪庭骑士 |
| 当前身份 | 从灰坑复活的未愈之伤宿主 |
| 武器 | 双手大剑“旧裁决剑” |
| 人物伤口 | 曾以为自己在净化邪恶，后来发现处决过无辜献祭者 |
| 主线目标 | 查清复活原因，阻止王国之伤继续吞噬活人 |
| 成长主题 | 从执行处决的人，变成阻止处决继续发生的人 |

洛温不是传统圣人。他的救赎不是洗白过去，而是一路补回自己曾经参与过的错误。未愈之伤让他变强，也让他逐渐非人化。

## 5. 主角模型方向

| 部位 | 设计要求 |
|---|---|
| 剪影 | 宽厚双手剑、低头站姿、破披风、重靴，整体沉重但不臃肿 |
| 头部 | 半遮面铁面甲，裂缝透出灰红微光 |
| 胸口 | 一道无法闭合的裂伤贯穿胸骨，战斗和剧情关键时发暗红光 |
| 武器 | 双手剑偏厚重，剑脊有断裂纹，剑格像残缺圣徽 |
| 服装 | 烧焦骑士长衣、旧锁甲、破披风，MUST NOT 使用夸张大肩甲堆砌压迫感 |
| 动画 | 普攻有拖刀感，重击有明显蓄势，冲刺带灰烬残影 |
| 受击表现 | SHOULD 有胸口裂伤闪烁和披风震动，强化“宿主”身份 |

## 6. 战斗与构筑方向

| 流派 | 定位 | 关键体验 |
|---|---|---|
| 重剑构筑 | 双手剑近战爆发 | 暴击、重击、处决、对精英增伤 |
| 灰焰构筑 | 中近距离范围清怪 | 灰焰范围伤害、燃烧地面、持续伤害 |
| 苦修构筑 | 低生命高风险输出 | 吸血、护盾、低生命增伤 |

第一版 SHOULD 先实现双手剑基础普攻、重击、一个范围技能、一个冲刺/保命技能。完整三流派 MAY 在后续版本扩展。

## 7. 主要角色

| 角色 | 定位 | 人物功能 | 模型/气质 |
|---|---|---|---|
| 洛温·阿什 | 主角 | 玩家角色，承担罪与救赎主题 | 灰烬骑士、双手大剑、破披风、胸口裂伤暗光 |
| 米拉 | 第一章幸存者 | 回城据点的第一个活人锚点，提醒主角仍可救人 | 短发、旧斗篷、手腕缠白灰布条，瘦小但眼神清醒 |
| 奥德里克 | 旧书记官 | 鉴定装备、解释词缀、记录遗境 | 盲眼老人，背铁皮书匣，手指有墨黑烧痕 |
| 瑟温修女 | 信仰动摇者 | 药剂、治疗、白焰圣堂真相线索 | 白焰修女服被烧灰，胸前圣徽裂开 |
| 埃蒙 | 第一幕反派 | 推动献祭仪式，引出白焰圣堂内部腐败 | 高瘦、长铁指套，衣摆沾满白灰 |
| 塞拉芬 | 白焰圣堂最高圣父 | 中后期核心反派，相信牺牲少数能保住世界 | 温和、克制、无血污，形成冷感压迫 |

## 8. 阵营与组织

| 名称 | 说明 | 视觉符号 |
|---|---|---|
| 白焰圣堂 | 旧王国官方教会，宣称白焰能净化污染 | 白色火焰、裂圣徽、干净但压迫的礼服 |
| 赦罪庭 | 白焰圣堂内部审判机构，负责抓捕和处决受污者 | 铁面具、审判锁链、封蜡文书 |
| 灰袍骑士 | 赦罪庭直属骑士部队，洛温曾经所属 | 灰袍、重剑、旧誓章 |
| 烛书记官 | 保存禁书、病历和献祭记录的学者群体 | 蜡封书册、铁皮书匣、烧痕手指 |
| 余民 | 灾后幸存者，不隶属于教会 | 补丁衣物、旧行囊、避难火盆 |
| 灰化者 | 被献祭后灵魂未散的人，不完全是怪物 | 灰皮、白灰结晶、重复祈祷动作 |
| 遗境回声 | 遗境里重复出现的记忆、怪物和仪式 | 灰焰、倒流灰烬、断裂圣徽 |

|---|---|---|

## 10. 第一章敌人

| 敌人 | 战斗功能 | 模型方向 | 掉落倾向 |
|---|---|---|---|
| 灰化村民 | 基础近战怪 | 佝偻、皮肤覆灰、手拿农具 | 普通装备、少量金币 |
| 捧火人 | 远程和自爆威胁 | 双手抱灰焰灯，死亡后灯裂爆燃 | 魔法装备、灰焰材料 |
| 白灰奴 | 慢速重甲怪，教学破防 | 面部被白灰硬壳封住，靠仪式本能移动 | 护甲、重武器 |
| 缝犬 | 快速扑击怪 | 多条村犬残肢缝合，背有白灰刺 | 靴子、移动相关词缀 |

## 11. 第一章 Boss

| 项目 | 设定 |
|---|---|
| 名称 | 赫尔曼，最后的村长 |
| 生前身份 | 烂骨村村长 |
| 人物罪责 | 曾主动向白焰圣堂献出村民名单，只为保住自己的家人 |
| 异化形态 | 全村死者的祈祷、怨恨和尸体被缝在赫尔曼身上，形成拖行的尸群 |
| 模型 | 上半身保留村长礼服和神父披肩，下半身是拖行尸群，背后挂断裂祭架 |
| 一阶段技能 | 重杖横扫、尸手抓取、召唤灰化村民 |
| 二阶段技能 | 胸腔裂开并喷出灰焰，场地出现灰焰带，召唤自爆捧火人 |
| 击败意义 | 终止烂骨村献祭循环，让死者停止重复受难，但无法复活他们 |
| 剧情产出 | 米拉成为据点 NPC；烂骨村坍缩为第一个遗境 |
| 玩法产出 | 开启裂口；掉落第一件剧情传奇双手剑部件 |

## 12. 第一章剧情骨架

| 节点 | 内容 |
|---|---|
| 开场 | 洛温从灰坑复活，胸口裂着未愈之伤，只记得自己曾被白焰圣堂处决 |
| 初遇怪物 | 洛温进入旧麦路，发现村民重复献祭当天的行为 |
| 真相线索 | 白木礼拜堂墙上刻着村民名字，证明烂骨村已被献祭不止一次 |
| 救下米拉 | 米拉是村里唯一活人，靠白灰掩盖气息活到现在 |
| 中段反转 | 玩家以为在救村，实际村民肉体早已死亡，灵魂被困在遗境循环 |
| Boss 前 | 赫尔曼请求“再登记一次，全村就能被赦免”，但这是遗境回声的谎言 |
| Boss 后 | 死者停止重复痛苦，米拉跟随洛温离开，烂骨村坍缩成第一个遗境 |
| 后续解锁 | 玩家通过裂口进入遗境刷装，通过深层进行爬层挑战 |

|---|---|---|

| 头盔 | 灰骑盔、白灰面甲、赦罪铁冠 | SHOULD 强调遮面、审判、残破圣物 |

| 项链 | 白焰残徽、断祷坠、余民护符 | SHOULD 和剧情、阵营、死者相关 |

## 14. 第一批词缀方向

| 词缀 | 作用 | 适合流派 |
|---|---|---|
| 攻击力提高 | 提升基础伤害 | 通用 |
| 暴击率提高 | 提高爆发稳定性 | 重剑构筑 |
| 暴击伤害提高 | 强化重击和处决收益 | 重剑构筑 |
| 灰焰伤害提高 | 提升灰焰技能伤害 | 灰焰构筑 |
| 持续伤害提高 | 强化燃烧和流血 | 灰焰构筑、苦修构筑 |
| 击杀恢复生命 | 提高清图续航 | 苦修构筑 |
| 低生命伤害提高 | 低生命构筑核心 | 苦修构筑 |
| 对精英伤害提高 | 提升 Boss 和遗境/深层精英战效率 | 通用 |
| 冷却缩减 | 提升技能循环 | 灰焰构筑、苦修构筑 |
| 移动速度提高 | 提升刷图效率 | 通用 |

## 15. 遗境与深层设定

| 类型 | 名称 | 用途 | 第一版范围 |
|---|---|---|---|
| 小秘境 | 遗境 | 刷装备、材料、词缀、裂片 | 1 套地图块、随机怪物组合、结尾精英 |
| 大秘境 | 深层 | 限时爬层，验证构筑强度 | 5 层原型，每层怪物强度递增，第 5 层 Boss |

遗境 SHOULD 让玩家快速获得掉落反馈。深层 SHOULD 让玩家感受到装备筛选和构筑优化的价值。第一版 MUST 先实现基础循环，不追求复杂随机生成。

## 16. 第一版非目标

| 非目标 | 说明 |
|---|---|
| 多职业 | 第一版只做洛温·阿什一个双手剑职业 |
| 完整主线 | 第一版只做第一章剧情 Demo |
| 完整套装系统 | 第一版可预留装备数据结构，但不实现套装 |
| 大量传奇装备 | 第一版只需要少量能改变打法的传奇效果 |
| 复杂交易系统 | 第一版不做拍卖、交易、多人经济 |
| 联机 | 第一版只做单机本地体验 |
| 精美最终美术 | 第一版允许使用占位图，但模型设计 MUST 按本文档约束推进 |
